# Snapledger 隱私權政策

生效日期：2026 年 3 月 18 日

本隱私權政策說明 [開發者名稱]（以下稱「我們」）在您使用 Snapledger（以下稱「本 App」）時，如何處理相關資料。

## 1. 適用範圍
本政策適用於您使用本 App 時所涉及的資料處理行為。

## 2. 我們處理哪些資料
就目前版本而言，本 App：

- 不要求您註冊帳號。
- 不會主動向您蒐集姓名、電子郵件、電話號碼或身分證明資料。
- 不會主動存取您的聯絡人、定位資訊、相機、麥克風、相簿、簡訊或通話紀錄。
- 未整合廣告 SDK、第三方分析 SDK 或崩潰回報 SDK。
- 會在 App 啟動時連線至 GitHub API 檢查是否有可用的新版本。
- 當您點選更新或下載按鈕時，會開啟 GitHub Release 或下載頁面。

上述與 GitHub 的連線過程中，GitHub 可能基於網路通訊所需處理您的 IP 位址、裝置或瀏覽器資訊、請求時間等技術資料。這些資料由 GitHub 依其隱私權政策處理。

## 3. 我們如何使用資料
我們處理資料的目的如下：

- 提供本 App 的基本功能與畫面顯示。
- 檢查是否有新版本可供您下載。
- 在您主動操作時，開啟外部更新頁面或下載頁面。
- 維持本 App 的基本穩定性與錯誤排查。

## 4. 資料分享與第三方服務
我們不會出售、出租或任意交易您的個人資料。

就目前版本而言，本 App 僅可能在下列情況與第三方服務互動：

- GitHub API：用於檢查最新版本資訊。
- GitHub 網站／下載頁：當您主動點選更新或下載連結時開啟。

第三方服務有其各自的隱私權政策與資料處理方式，我們建議您一併參閱。

## 5. 資料保存與刪除
就目前版本而言，我們不在自有伺服器上建立使用者帳號資料庫，也不在自有伺服器上保存您的個人記帳資料。

若您透過電子郵件或其他方式聯絡我們，我們可能保留您提供的聯絡內容，以便回覆問題、處理請求或符合法律義務；在目的完成後，我們會於合理期間內刪除或停止使用該等資料。

## 6. 資料安全
我們採取合理的技術與管理措施保護資料安全，並盡量避免未經授權的存取、使用或揭露。不過，任何網路傳輸或電子儲存方式都無法保證百分之百安全。

## 7. 兒童隱私
本 App 並非專門提供給未滿 13 歲兒童使用。我們不會故意蒐集兒童的個人資料。如您認為兒童已提供相關資料，請透過下方聯絡方式通知我們。

## 8. 您的權利
在適用法律允許的範圍內，您得就您的個人資料提出查詢、更正、刪除或其他相關請求。您也可以透過聯絡方式向我們詢問本政策內容。

## 9. 隱私權政策更新
若本 App 功能、資料處理方式或法規要求有所變更，我們可能更新本政策。更新後版本將公布於 [網站網址]，並自公布時或公告載明之日期起生效。
